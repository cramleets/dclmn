== Description ==
Eventful is an add-on for [The Events Calendar](https://wordpress.org/plugins/the-events-calendar/) that lets you showcase your events using modern, responsive layouts designed for real-world event websites.

Display events in sliders, carousels, grids, or lists with an optional filter bar, and customize the look to match your website — no coding required.

Built specifically for The Events Calendar, Eventful works seamlessly with your existing events and categories while giving you flexible display options that look great on all devices. With support for popular page builders like Elementor and Gutenberg, Eventful makes it easy to highlight upcoming events, workshops, conferences, and community activities in a clean, professional, and engaging way.

Integrating Appsero SDK **DOES NOT IMMEDIATELY** start gathering data, **without confirmation from users in any case.**

== Installation ==
1. Install and activate [The Events Calendar](https://wordpress.org/plugins/the-events-calendar/)
2. Upload the `eventful` plugin folder to the `/wp-content/plugins/` directory or install it via Plugins > Add New
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to **Eventful > Add New Layout** to create your first event showcase
5. Use the generated shortcode or Gutenberg block to display it on your site

== Frequently Asked Questions ==
= What is Eventful – Events Showcase for The Events Calendar? =

Eventful is a WordPress plugin that extends [The Events Calendar](https://wordpress.org/plugins/the-events-calendar/) plugin and allows you to display events in beautiful layouts such as grid, carousel, slider, and list views. It helps you create modern event showcases with filtering and customization options.

= Do I need The Events Calendar plugin to use Eventful? =
Yes. Eventful works as an **addon for The Events Calendar plugin**. You must install and activate [The Events Calendar](https://wordpress.org/plugins/the-events-calendar/) before using Eventful.

= Can I display events in a grid layout? =  
Yes. Eventful allows you to display events in **grid layouts**, making it easy to create modern event showcases for conferences, meetups, webinars, and other events.

= Can I create an event carousel or slider? =  
Yes. Eventful includes **carousel and slider layouts** so you can showcase upcoming events in a dynamic and interactive way.

= Can I display events in a list layout? =
Yes. Eventful includes a minimal **list layout** that allows you to display events in a clean and simple format. This layout is available in the free version and works great for showing upcoming events in an organized list.

= Can visitors filter events by category or tag? =
Yes. Eventful allows visitors to **filter events by categories, tags, venues, organizers, and dates**, helping users find the events they are interested in quickly.

= Does Eventful work with Elementor? =
Yes. Eventful works perfectly with **Elementor**, allowing you to place event showcases anywhere on your page using shortcodes or widgets.

= Can I display events using a shortcode? =
Yes. Eventful provides **shortcodes** so you can easily display event layouts inside posts, pages, or custom sections of your website.

= Does Eventful support Gutenberg block editor? =
Yes. Eventful works with the **WordPress Gutenberg editor** and allows you to embed event showcases within block-based layouts.

= Can I show only upcoming events? =
Yes. Eventful allows you to display **upcoming events, past events, or specific event categories** depending on your configuration.

= Can I customize the event layout design? =
Yes. Eventful provides multiple customization options including layout styles, colors, event metadata display, and filtering options.

= Does Eventful support responsive layouts? =
Yes. All Eventful layouts are **fully responsive** and work perfectly on desktop, tablet, and mobile devices.

= Is Eventful compatible with most WordPress themes? =
Yes. Eventful is built following WordPress coding standards and works with most modern WordPress themes.

= Is there a Pro version of Eventful? =
Yes. [Eventful Pro →](https://wpeventful.com/) includes additional layouts, advanced filtering, more customization options, and enhanced event display features. [Compare Free vs Pro →](https://wpeventful.com/free-vs-pro/)

You can get support through the [WordPress.org support forum](https://wordpress.org/support/plugin/eventful/) for the free version or through our website for [premium support](http://themeatelier.net/create-support-ticket/).

== Changelog ==
= 2.2.5 - 06 July 2026 =
* Fixed: Content with Limit (characters/words) no longer outputs raw markup such as block comments (`<!-- wp:tribe/event-… -->`) or unclosed tags (`<img class="align…`). Truncation now counts visible, rendered text, cuts on word boundaries, is multibyte-safe, and only appends the ellipsis when content is actually shortened — consistent across Grid, List, Masonry, Timeline, and Table layouts.

= 2.2.4 - 24 June 2026 =
* Fixed: Event Meta Fields → Venue — City, State, Country, and Postal Code fields now display correctly in layouts when selected. Previously these fields were retrieved from the database but not included in the output.

= 2.2.3 - 13 June 2026 =
* Added: Generator tabs and sub-tabs now remember the last selected tab and restore it when you return.

= 2.2.2 - 23 May 2026 =
* Fixed: Activation redirect used `wp_safe_redirect() + exit` inside the `activated_plugin` hook, causing activation errors when activated programmatically (e.g. WP-CLI or automated tools).
* Fixed: DB update script included via relative path, which could fail on servers where PHP resolves includes from the WordPress root instead of the plugin directory.

= 2.2.1 - 19 May 2026 =
* Added: Timeline global colors option.

= 2.2.0 - 19 May 2026 =
* Added: Vertical Timeline layout with month/year separator, timeline line and dot color controls, and date badge color.
* Added: Carousel Style sub-picker (Standard available free; Ticker, Center, and Multi Row available in Pro).
* Fixed: Load More pagination now correctly continues the left/right alternation for the Timeline layout when events per page is odd.
* Updated: DB migration to convert legacy `event_type_time` filter data to the unified `eventful_event_type` meta key.
* Updated: Admin panel CSS for sub-preset layout pickers.

= 2.1.23 - 05 May 2026 =
* Fixed: `add_filter` incorrectly used instead of `add_action` for the `init` hook, preventing post type registration in some edge cases.
* Fixed: `add_shortcode()` called with unsupported extra arguments, causing PHP warnings.
* Fixed: Deprecated `current_time('timestamp')` replaced with `time()` for WordPress 6.x compatibility.
* Fixed: Update script included via a relative path, which could cause activation errors depending on server configuration.
* Fixed: `wp_safe_redirect()` incorrectly wrapped in `exit(wp_kses_post(...))`, causing malformed redirect output.
* Fixed: Double `json_decode()` with `stripslashes()` in the import function, which caused import failures on PHP 8.
